### Setup Instruction

1. Clone the project.

```
    git clone https://github.com/ChinmayKaitade/Learning-Management-System-Client
```

2. Move into the directory.

```
    cd Learning-Management-System-Client
```

3. Install dependencies.

```
    npm i
```

4. Run the server.

```
    npm run dev
```
